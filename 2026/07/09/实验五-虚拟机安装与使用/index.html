    实验五 虚拟机安装与使用

    Posted on 2026-07-09 Edited on 2026-07-10

    报告人： 杨欣雨 25051503

    实验五：虚拟机安装与使用

    1. 实验目的

    本次实验旨在通过 Oracle VirtualBox 虚拟化软件，掌握虚拟机的安装、配置与使用方法。具体包括：在虚拟机中安装 Ubuntu Server 24.04 LTS 操作系统，配置 NAT 与仅主机（Host-Only）双网卡模式，设置静态 IP 地址，并通过克隆技术快速构建由多台虚拟机组成的虚拟化实验环境，最终验证虚拟机之间的网络连通性。

    2. 实验环境

    项目 详细信息
    宿主机操作系统 Windows 11
    虚拟化软件 Oracle VirtualBox 7.0+
    虚拟机操作系统 Ubuntu Server 24.04.4 LTS
    虚拟机内存 2048 MB
    虚拟机硬盘 20 GB（动态分配）
    虚拟机网络 NAT + 仅主机（Host-Only）双网卡
    IP 规划 template: 192.168.56.101
    node1: 192.168.56.102
    node2: 192.168.56.103
    node3: 192.168.56.104

    3. 实验过程

    步骤一：下载并安装 VirtualBox

    图：VirtualBox 下载页面
    从 VirtualBox 官网下载 Windows 版本安装包，双击运行，按默认配置完成安装。安装完成后，打开 VirtualBox 主界面，确认界面显示正常。

    图：VirtualBox 主界面

    步骤二：下载 Ubuntu Server ISO 镜像

    访问 Ubuntu 官网，下载 Ubuntu Server 24.04.4 LTS 的 ISO 镜像文件，保存至本地磁盘。


    图：Ubuntu Server 24.04.4 LTS 下载页面

    步骤三：创建虚拟机（模板机）

    在 VirtualBox 中点击“新建”，配置如下参数：

    • 名称：ubuntu-template
    • 文件夹：E:\VirtualMachines\ubuntu-template
    • ISO 映像：选择已下载的 Ubuntu Server 24.04.4 LTS ISO
    • 类型：Linux，版本：Ubuntu (64-bit)

    • 内存：2048 MB
    • 硬盘：20 GB，动态分配，VDI 格式

    图：虚拟机创建配置界面

    步骤四：安装 Ubuntu Server 操作系统

    启动虚拟机，按照以下步骤完成系统安装：

    1. 选择语言：English
    2. 键盘布局：English (US)
    3. 网络配置：保持默认（DHCP）
    4. 代理配置：留空
    5. 软件源：保持默认

    6. 存储配置：选择 Use an entire disk
    7. 确认分区方案后选择 Continue

    8. 设置用户信息：
      • 用户名：vboxuser
      • 密码：ABCD1234
    9. 选择安装 OpenSSH Server（按空格键选中）

    10. 等待安装完成，选择 Reboot Now

    图：安装完成后登录界面

    步骤五：配置双网卡

    安装完成后，关闭虚拟机。在 VirtualBox 中右键 ubuntu-template → 设置 → 网络：

    • 网卡1：启用，连接方式为 NAT（用于上网）
    • 网卡2：启用，连接方式为 仅主机（Host-Only）网络（用于与宿主机及其他虚拟机通信）

    图6：VirtualBox 双网卡具体信息

    步骤八：克隆虚拟机

    关闭模板机后，在 VirtualBox 中右键 ubuntu-template → 选择“复制”：

    • 名称：node1
    • 路径：E:\VirtualMachines\node1
    • 克隆类型：完全复制
    • MAC 地址策略：为所有网卡重新生成 MAC 地址

    同样的方法复制出 node2 和 node3。


    图：VirtualBox 克隆配置界面
